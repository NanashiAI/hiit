# HIIT タイマー

アプリのインストール不要で、ブラウザだけで使えるHIITタイマーです。  
タバタ式（20秒ワーク／10秒レスト × 8セット）に対応しています。

👉 **[タイマーを開く](https://nanashiai.github.io/hiit/)**

---

## こんな人向け

- 手軽にHIITトレーニングを始めたい人
- スマホでそのまま使えるシンプルなタイマーが欲しい人
- アプリをインストールしたくない人

---

## 主な機能

| 機能 | 説明 |
|---|---|
| タバタ式タイマー | 20秒ワーク／10秒レスト × 8セット |
| 画面スリープ防止 | 実行中はスマホ画面が消えない（Wake Lock API） |
| フェーズ切り替え通知 | バイブレーションでワーク／レストを通知 |
| PAUSE / RESUME | 途中で一時停止・再開が可能 |
| RESET | いつでも最初からやり直せる |

---

## 使い方

1. 上のリンクからタイマーを開く
2. **START** を押してトレーニング開始
3. 画面の色でフェーズを確認（赤＝ワーク / 緑＝レスト）
4. 8セット終了で自動終了
